# Load Testing API, REST: Agent.update
Updates the specified agent.
 

 
## HTTP request {#https-request}
```
PATCH https://loadtesting.{{ api-host }}/loadtesting/api/v1/agent/{agentId}
```

--- | ---
agentId | <p>Required. ID of the agent to update.</p>
